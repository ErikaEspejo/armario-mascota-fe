# Iconos PWA

Este directorio debe contener los iconos de la PWA.

## Iconos Requeridos

- `icon-192x192.png` - Icono de 192x192 píxeles
- `icon-512x512.png` - Icono de 512x512 píxeles

## Nota

Los iconos actuales son placeholders. Reemplázalos con los iconos reales de la aplicación antes de desplegar a producción.

Puedes generar iconos usando herramientas como:
- [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
